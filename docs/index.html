Download

Usage

  1. Enter your username in top field (along with the website name)
  2. Enter your master password below
  3. Copy the hash generated into the password field. Next time the same password will be generated

How does it work?

  1. Each combination of username, website and master password is hashed to make a UNIQUE password
  2. Even if the password is leaked the person who gets the password cannot get your master password

Advantages

  1. You no longer have to click
  2. You don't have to carry around a file unlike other password managers

No data is collected from the application.

Windows Defender Smartscreen might show that it "protected your pc", but this happens because haven't signed my Windows application with a code signing certificate. A code signing certificate is of two types:

  1. OV certificates: These are the waste of money that don't even disable the warnings.
  2. EV certificates: A waste of money ($755) per YEAR issued only to organisations (to enable loss of budget maybe).
